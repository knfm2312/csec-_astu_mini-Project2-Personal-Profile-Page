# Personal Portfolio Website

This is my personal portfolio website. It shows my profile, skills, and contact information.

## Features
- Responsive design for all screen sizes  
- Profile section with bio and image  
- Skills section with gradient cards  
- Contact form  
- Background image and hover effects  

## Files
- `index.html` – main webpage  
- `style.css` – styling for the page  
- `images/` – folder for profile and background images  

## How to Use
1. Download or clone this project.  
2. Open `index.html` in your browser.  
3. You can edit the text, colors, or images in the files.
